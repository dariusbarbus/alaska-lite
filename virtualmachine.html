Setting up Alaska Lite 11 in VMware Workstation Player

In this guide, we'll show you how to install and test the Alaska Lite 11 ISO in a VM.
This guide uses version 16 of VMware Software.

1. Download VMware Workstation Player

We must download the VMware Workstation Player from the VMware website, you can follow this link or you can search for VMware Workstation Player, the software is free to use for non-commercial use.

Once you have downloaded the file, install it by running the .exe file. Now that the software is up and running is time to follow the next step.

2. Create a Virtual Machine

With the program open, you must follow the next steps:

  1. Create a New Virtual Machine
  2. Select disk image file (iso) and browse to the Alaska Lite 11 ISO
  3. Make sure that Guest operating system is Microsoft Windows and the version is Windows 10 and later x64
  4. Give it any name and the location (default location is recommend)
  5. You can leave this page alone, default disk size of 60 GB is fine
  6. Select Customize Hardware
  7. In the Memory tab, where it says 2048 MB, change it to 4096 (Change it to a higher value if you know that you are doing)
  8. Click Close and then Finish

Now it will apear as a new VM in the menu, now we must procede to enable TPM in the configuration file of the VM
