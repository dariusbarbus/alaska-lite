Why Alaska Lite 11

We all use Windows 11, either for documents, games, graphic design, or just browsing the internet, and it's a good experience, but, Microsoft is pushing more and more services and bloatware that 99% of user won't ever use, so we made it our task to create a version of Windows that works the way it should. This version is not meant to be the absolute lightest but the lightes while usable for end users With limited knowledge about software.

Over the View

  • Tweaks

    Using the Registry Editor we were able to modify different parts of the OS to enhance the experience. With the classic right click menu, the ribbon in Explorer and much more.

  • Performance

    With fewer proceses and services running in the background, this is a much faster and lighter Windows 11 that doesnt sacrifice usabilty for a few less MBs of RAM.

  • Passion Project

    This is a passion project created by Dario Simpson in order to have a lightweight, fast and usable OS for the everyday user.

It can be hard to trust

We know it can be hard to trust a OS from the internet, feel free to install it in a VM, We recommend VMware Player, and check everything to make sure you are comfortable using this OS. All of the changes to the OS made are publicly available in the following link. Also you can use a firewall like Glasswire, or TinyWall to check if there is a cryto or any kind of funny busineses running in the background, of which we guarantee there is not, but, if you find one, please submit a issue inmidiatly.
