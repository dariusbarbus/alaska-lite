Why Alaska Lite 11

We all use Windows 11, either for documents, games, graphic design, or just browsing the internet, and it's a good experience, but, Microsoft is pushing more and more services and bloatware that 99% of user won't ever use, so we have made it out task to create a version of Windows that works the way it should. This version is not meant to be the absolute lightest, but the lightest while usable for end users with limited knowledge about software.

Over the View

  • Tweaks

    Using the Registry Editor, we were able to modify different parts of the OS. With the classic right click menu, the ribbon in Explorer, no fast start-up and more.

  • Performance

    With fewer processes and services running in the background, this is a much faster and lighter Windows 11 that doesn't sacrifice usability for a few less MB's of RAM.

  • Passion Project

    This is a passion project created by Dario Simpson in order to have a lightweight, fast and usable OS for the everyday user.

Let's Show the Numbers

Claims mean nothing without proof, here are the current statistics of Alaska Lite 11 vs Windows 11

  • 100Processes
  • 1100Megabytes of RAM
  • < 10 GB Install Size
  • No phoneNo telemetry
  • Actually Fast

At idle we can see it's using less than 100 processes, while Windows 11 uses about 150. The RAM usage has decrease exponentially from more than 2 GB in a standard Install to almost 1 GB, that's right there with a Linux install. The installation size has been slash in half, literary, Windows 11 takes about 18 to 19 GB in a standard install, now is less than 10 GB. There is no phoning home from the OS with telemetry and ads removed. This leaves us with an actually fast and lightweight OS for everyday use.

It can be hard to trust

We know it can be hard to trust an OS from the internet, feel free to install it in a VM, We recommend VMware Player, and check everything to make sure you are comfortable using this OS. Also, you can use a firewall like Glasswire, or TinyWall to check if there is a cryto or any kind of funny busineses running in the background, of which we guarantee there is not, but, if you find one, please submit a issue inmidiatly.
