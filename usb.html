Make a bootable USB with Alaska Lite 11

Lets see how to set up Alaska Lite 11 to boot from a USB so you can install it to your machine

1. Donwload and run Rufus

Before we start, make sure you have an 8 GB or larger USB drive that will be formatted. Now, just head to the Rufus website and download the latest portable version, this version won't install to your system because is for a one time use.

2. Run the software

The software will autodetect the USB, in case it's showing another drive, just click de dropdown menu and select the correct one. Select the ISO file using the SELECT button and then click START, it might give a couple of warnings but just skip them and let the software finish coping the ISO into the USB drive, and after that you are ready to boot from it and install Alaska Lite 11.
This guide won't show you how to boot into your BIOS and select the USB drive, there are plenty of tutorials, just search for your motherboard manufacture followed by BIOS key.
